Get Anuvaad AI running on your laptop in just a few simple steps!

================================================================================
                           PREREQUISITES
================================================================================

Install these on your system first:

1. Python 3.10 or higher
   Download: https://www.python.org/downloads/
   Verify: python --version

2. Node.js 18 or higher
   Download: https://nodejs.org/
   Verify: node --version

3. FFmpeg (Required for video/audio processing)
   Windows: https://ffmpeg.org/download.html
   Mac: brew install ffmpeg
   Linux: sudo apt-get install ffmpeg
   Verify: ffmpeg -version

4. VS Code (Recommended)
   Download: https://code.visualstudio.com/

================================================================================
                     QUICK START (3 SIMPLE STEPS)
================================================================================

STEP 1: Get API Keys
--------------------
You need API keys to use the AI features:

a) ElevenLabs (for AI voices):
   - Sign up at: https://elevenlabs.io/
   - Get API key: https://elevenlabs.io/app/settings/api-keys

b) Google Gemini (for translation):
   - Get API key: https://ai.google.dev/


STEP 2: Install Dependencies
-----------------------------
Open terminal in VS Code (Ctrl+` or Cmd+`) and run:

a) Create Python virtual environment:
   python -m venv venv

b) Activate virtual environment:
   Windows:   venv\Scripts\activate
   Mac/Linux: source venv/bin/activate

c) Install Python packages:
   pip install -r app_requirements.txt


STEP 3: Configure API Keys
---------------------------
a) Copy the example file:
   Windows:   copy .env.example .env
   Mac/Linux: cp .env.example .env

b) Edit .env file and add your API keys:
   - ELEVENLABS_API_KEY=your_actual_key_here
   - GEMINI_API_KEY=your_actual_key_here
   - JWT_SECRET_KEY=any_random_string_here

   Tip: Generate JWT secret with:
   python -c "import secrets; print(secrets.token_hex(32))"

================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: ".env file not found" error
Solution: Make sure you created .env file from .env.example
         cp .env.example .env

Problem: "Module not found" error  
Solution: Activate virtual environment and install dependencies:
         pip install -r requirements.txt

Problem: "FFmpeg not found" error
Solution: Install FFmpeg and ensure it's in your system PATH

Problem: "Port already in use" error
Solution: Stop any apps using ports 5000 or 5001, or restart your computer

Problem: Frontend dependencies installing slowly
Solution: This is normal the first time. It might take 2-5 minutes.

Problem: API key errors when using features
Solution: Check your .env file has correct API keys with no extra spaces
